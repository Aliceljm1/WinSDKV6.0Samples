Generic manage code-based Winform application

The RMWINFORMAPP sample is a generic managed code-based winform application that
provides only limited (text input) notepad functionality.


This application showcases the minimal level of functionality required to 
enable a managed code winform application to be shutdown and restart by Restart 
Manager aware applications.


Build Instructions:

Load the project into visual studio 2005 and compile.

Notes:
You could port the project solution to vs2003 if desired
