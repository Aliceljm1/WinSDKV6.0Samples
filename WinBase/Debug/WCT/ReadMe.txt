========================================================================
    CONSOLE APPLICATION : WctEnum Project Overview
========================================================================

This project demonstrates the usage of the Wait Chain Traversal (WCT) API.

The sample code shows how to retrieve the wait chain for all threads
in the system.  For best results this sample should be run from an
elevated command prompt.

File overview:

WctEnum.cpp
    This is the main application source file.

WctEnum.vcproj
    This is the Visual Studio main project file for this project.

WctEnum.sln
    This is the Visual Studio solution file

makefile
    Makefile for nmake.exe
