Using ADO 2.7 with OpenXML

Sample Overview

OpenXML is a new feature of Microsoft SQL Server 2000. It allows XML to be passed into a SQL Stored Procedure and processed by T-SQL commands as if it was a SQL Rowset.

Setup

Create the OpenXML Stored Procedure.
Using SQL Server Query Analyzer, open SP_OpenXML_Example.sql and execute the contents. The file also contains test code to ensure your SP is running properly.

Running the Sample:

Follow the instructions below to run this sample.
Type http://localhost/ADOSamp/ADOSQL2K/OpenXML.asp in your browser,
or click here to run the OpenXML Sample

System Requirements

This application requires the following system components to run.
  • Microsoft Windows NT 4.0 or Windows 2000
  • Microsoft Internet Information Services version 4.0 or 5.0
  • Microsoft Visual C++ version 6.0 or newer
  • SQL Server 2000 or newer
