Using ADO 2.7 with Select ... For XML Queries

Sample Overview

Prior versions of ADO retrieved data in the form of a Recordset object. Beginning with version 2.7, when ADO is used with Microsoft SQL Server 2000, results can also be retrieved directly from SQL Server in the form of an XML document. This is done by appending the "For XML" clause to the end of the SQL Select statement. For more infomation on For XML queries, see the ADO 2.7 Documentation.

Setup

There is no additional setup required to run this sample.

Running the Sample

Follow the instructions below to run this sample.
Point your browser to http://localhost/adosamp/adosql2k/forxml.asp, or click here to run the For XML Sample.

System Requirements

This application requires the following system components to run.
  • Microsoft Windows NT 4.0 or Windows 2000
  • Microsoft Internet Information Services version 4.0 or 5.0
  • Microsoft Visual C++ version 6.0 or newer
  • SQL Server 2000 or newer
