Using ADO 2.7 with XPath Queries

Sample Overview

Beginning with version 2.7, the OLE DB Provider for SQL Server 2000 can now execute queries specified using the new XPath command language proposed by the W3C. This sample demostrates how to pass in an XPath command via ADO, using this new dialect.

Setup

In XPath.asp, verify that the "Base Path" property points to the directory containing the Orders.xml Mapping Schema. By default it is set as follows:
adoCmd.Properties("Base Path") = "C:\INETPUB\WWWROOT\ADOSamp\ADOSQL2K"

Running the Sample:

Follow the instructions below to run this sample.
Point your browser to http://localhost/ADOSamp/ADOSQL2K/XPATH.asp?CompanyName="Tortuga Restaurante",
or click here to run the XPath Sample

System Requirements

This application requires the following system components to run.
  • Microsoft Windows NT 4.0 or Windows 2000
  • Microsoft Internet Information Services version 4.0 or 5.0
  • Microsoft Visual C++ version 6.0 or newer
  • SQL Server 2000 or newer
