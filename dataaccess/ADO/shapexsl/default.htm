Sales Analysis Using XML and XSL

Sample Overview

This sample has six key pages:
  • ShapeXSL.htm - The start page of the sample - a simple HTML form.
  • OrdersProcess.asp - This page is the target of the form's POST request. It processes the form variables and submits an SQL query to one of the following two pages (depending on the user input)
  • OrdersToTableServer.asp - This page takes the incoming SQL query, submits it to SQL Server, and gets the XML results back in an ADO Stream object. It runs this XML through an XSL transform using server-side script to create an HTML table, and outputs this table to the returned page.
  • OrdersToTableClient.asp - This page takes the incoming SQL query, submits it to SQL Server, and gets the XML results back in an ADO Stream object. It sends this XML back as a Data Island to the client browser, which then uses client-side script to run the XML through an XSL transform to create an HTML table.
  • TableFromXMLHier.xsl - XSL transformation of a SQL Server 2000 FOR XML AUTO format two-level XML hierarchy into an HTML table.
  • Default.htm - This page

Setup

This sample application assumes you have already created the ADOSamp VRoot as specified on the ADOSamp home page.

Running the Sample:

Enter the following URL: http://localhost/adosamp/shapexsl/shapexsl.htm in your browser or click here to run the Sample.
Fill out the form, and click the 'Get Data' button.

System Requirements

This application requires the following system components to run.
  • Microsoft Windows NT 4.0 or Windows 2000
  • Microsoft Internet Information Services version 4.0 or 5.0
  • SQL Server 2000
