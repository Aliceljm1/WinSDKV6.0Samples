(1) You must have Visual Studio Whidbey installed
(2) Launch the solution by clicking on OCSPSample.sln file.
(3) In the Solution Explorer Tab right click on References and Select 'Add Reference'..
(4) In the References Property Page that pops up select the 'Browse' Tab
(5) Add the "Microsoft.CertificateAdministration.Interop.dll" by typing 
    in "%systemroot%\system32\Microsoft.CertificateAdministration.Interop.dll" in the 'File' field
    at the bottom of the 'Browse' page
