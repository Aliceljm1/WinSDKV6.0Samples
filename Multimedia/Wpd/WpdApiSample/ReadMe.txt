========================================================================
    CONSOLE APPLICATION : WpdApiSample Project Overview
========================================================================

This file contains a summary of what you will find in each of the files that
make up your WpdApiSample application.


WpdApiSample.vcproj
    This is the main project file for VC++ projects generated using an Application Wizard. 
    It contains information about the version of Visual C++ that generated the file, and 
    information about the platforms, configurations, and project features selected with the
    Application Wizard.

WpdApiSample.cpp
    This is the main application source file.

ContentEnumeration.cpp
	This file demonstrates how to enumerate content on a portable device.
	
ContentProperties.cpp
	This file demonstrates how to read/write properties for content on a portable device.
	
ContentTransfer.cpp
	This file contains examples of how to transfer content on/off a portable device.

DeviceCapabilities.cpp
	This file contains examples of how to query the capabilities of a portable device.
	
DeviceEnumeration.cpp
	This file contains examples of how to enumerate portable devices.
	
DeviceEvents.cpp
	This file contains examples of how to register/unregistert for portable device events.


/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named WpdApiSample.pch and a precompiled types file named StdAfx.obj.

CommonFunctions.h
	This file contains function prototypes which are shared across multiple .cpp source files
