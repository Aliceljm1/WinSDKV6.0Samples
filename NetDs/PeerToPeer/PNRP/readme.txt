Description
-----------
The PNRP Sample demonstrates the use of a subset of the Peer to peer APIs centered around the Peer Name Resolution Protocol

While using this application, one caveat to keep in mind is that you cannot resolve a name that is registered within the same process

How to Build
------------
nmake (builds and runs on all platforms)

How to Run
----------
Type pnrp.exe from ...\PeerToPeer\PNRP\LH_DEBUG\
Follow the menu options


Platforms supported
-------------------
Windows Vista (except Starter Edition)
Windows XP SP2
