Description
-----------
The Graphchat sample demonstrates the use of a subset of the Peer to peer APIs centered around Graphing, People Near me and Application Invite. It shows a multiparty chat application into which you can invite people located nearby

How to Build
------------
nmake

How to Run
----------
Type graphchat.exe from ...\PeerToPeer\GraphChat\LH_DEBUG\
Just use the UI!


Platforms supported
-------------------
Windows Vista (except Starter Edition)
